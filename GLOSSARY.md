# Glossary / Глоссарий — RU ↔ EN

> **Locked terminology.** Канонический перевод терминов ноомарксизма. Используй ровно эти соответствия в любых англоязычных материалах и переводах «на лету». Цель — не дать переводу исказить теорию.
>
> **Source of truth — Russian.** Русские документы и [noomarxism.ru](https://noomarxism.ru) первичны. Этот глоссарий лишь фиксирует, как термины звучат по-английски.

## Core class & revolution

| RU | EN (canonical) | Note |
|----|----------------|------|
| Ноомарксизм | **Noomarxism** | Not "neo-Marxism" (Frankfurt School). A distinct paradigm. |
| Когнитариат | **Cognitariat** | The new vanguard class. Adjective: *cognitarian*. |
| Когнитарий | **Cognitarian** (noun, member of the cognitariat) | |
| Пролетариат | **Proletariat** | Carrier of alienated labor. |
| Когнитарная революция | **Cognitarian revolution** | Automates *cognitive* labor. ⚠️ Not "cognitive revolution" (Harari sense). Site URL slug is `cognitive-revolution`, but the precise term is *cognitarian*. |
| Креативный класс | **Creative class** | R. Florida's concept — ≠ cognitariat (lacks revolutionary potential). |

## Labor & alienation

| RU | EN (canonical) | Note |
|----|----------------|------|
| Труд | **Labor** | |
| Разделение труда | **Division of labor** | Four types: physical, vertical, class, cognitive. |
| Отчуждение | **Alienation** | |
| Отчуждённый труд | **Alienated labor** | Proletariat. |
| Неотчуждённый труд | **Non-alienated labor** | Cognitariat. *Not* "partially alienated". |
| Всеобщий труд | **General labor** | Marx's *allgemeine Arbeit*. |
| Орудийное усиление | **Tool-based augmentation** | How cognitive division of labor is overcome (via AI), not by rotation. |
| Профессиональный кретинизм | **Occupational idiocy** | Ilyenkov / Marx. |
